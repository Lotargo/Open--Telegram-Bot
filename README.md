# AI Portfolio Bot

Telegram бот-визитка, который работает как "умный секретарь". Бот использует LLM (Groq) для ответов на вопросы о услугах и ценах, а также для оформления заявок от клиентов.

## Функционал

*   **Умный секретарь:** Отвечает на вопросы, используя прайс-лист из базы данных.
*   **Заявки:** Собирает данные клиента и формирует карточку заявки.
*   **Подтверждение:** Требует явного подтверждения от пользователя перед отправкой админу.
*   **Rate Limiting:** Защита от спама (ограничение частоты сообщений).
*   **Мультитенантность:** Поддержка множества пользователей одновременно.

## Установка и запуск

1.  **Создайте .env файл:**
    Скопируйте пример из инструкции ниже и вставьте свои ключи.
    ```env
    BOT_TOKEN=ваш_токен_бота
    ADMIN_GROUP_ID=
    GROQ_API_KEY=ваш_ключ_groq
    MONGO_URI=mongodb://mongo:27017/portfolio_bot
    ```

2.  **Запуск через Docker Compose:**
    ```bash
    docker-compose up --build -d
    ```

3.  **Настройка Админа:**
    *   Добавьте бота в группу или напишите ему лично (туда, куда хотите получать заявки).
    *   Напишите команду `/set_admin`.
    *   Бот пришлет ID чата. Вставьте этот ID в `.env` файл в поле `ADMIN_GROUP_ID`.
    *   Перезапустите бота: `docker-compose restart bot`

## Разработка

*   Исходный код находится в папке `src/`.
*   База данных MongoDB хранит данные в volume `mongo_data`.
